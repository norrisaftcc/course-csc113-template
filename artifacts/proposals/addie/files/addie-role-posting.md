---

## What is ADDIE Reimagined?

ADDIE Reimagined evolves the classic instructional design framework for AI-era realities:

| Classic ADDIE | ADDIE Reimagined |
|---------------|------------------|
| Analyze once, upfront | **A**gile—continuous discovery |
| Design completely before building | **D**esign—emergent architecture |
| Develop to specification | **D**evelopment—AI-assisted |
| Implement when "done" | **I**terative—sprint delivery |
| Evaluate at the end | **E**nhancement—continuous improvement |

You'll work in 2-week sprints alongside a software development faculty member and visual designer, using GitHub-based workflows and AI tools to accelerate curriculum creation.
