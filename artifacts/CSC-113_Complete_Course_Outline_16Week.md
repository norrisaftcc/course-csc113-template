**Credits**: 3 (2 hours class + 2 hours lab)  
**Prerequisites**: None  
**Target Audience**: Ages 18-65, mixed technical abilities  

---

## COURSE OVERVIEW

### Official Description
"This course provides a survey of artificial intelligence and machine learning. Topics include the history, development, and current applications of artificial intelligence and machine learning. Upon completion, students should be able to demonstrate general artificial intelligence and machine learning concepts."

### Course Philosophy
**"We're not teaching them to code. We're teaching them to learn."**

This course uses AI tools as the context for teaching second-order skills:
- **Documentation** - Clear communication of process and results
- **Resilience** - "Failure is just exercise" - iteration over perfection
- **Growth Mindset** - Learning from mistakes, celebrating progress
- **Critical Thinking** - Questioning AI outputs, understanding limitations
- **Professional Workflow** - GitHub-first from day one

## TWO PARALLEL TRACKS (Beginning Week 7)

### Prompt Masters Track
**Focus**: Natural language AI collaboration, no-code tools, prompt engineering mastery

### Code Builders Track
**Focus**: Programming with AI assistance, technical implementation, API integration

### Track Convergence
Both tracks collaborate on community projects in Weeks 11-16, creating portfolio-ready artifacts that demonstrate equivalent competency through different methods.
