# CSC-113 AI Fundamentals: Complete 16-Week Outline

**Version:** 1.0
**Last Updated:** 2026-01-01
**Structure:** 8 Modules × 2 Weeks Each

---

## Course Philosophy

> "Failure is just exercise. Delete the branch and try again."

This course teaches AI fundamentals through professional practice. Students learn:
- GitHub-first workflow from day 1
- Process documentation over perfect products
- Critical evaluation of AI tools
- Both technical and no-code approaches (equal rigor)

**Two Tracks (Weeks 7+):**
- **Code Builders**: Programming, APIs, technical implementation
- **Prompt Masters**: No-code tools, advanced prompting, business applications

## Assessment Framework

### Four Categories (25% each)

1. **AI Partnership Quality**
   - Prompt sophistication and iteration
   - Appropriate tool selection
   - Evidence of learning from AI interactions
   - Critical evaluation of outputs

2. **Problem-Solving Process**
   - Clear problem definition
   - Systematic approach
   - Testing methodology
   - Learning from failures

3. **Professional Communication**
   - Documentation clarity
   - GitHub portfolio quality
   - Peer review participation
   - Presentation skills

4. **Critical Thinking & Ethics**
   - Questioning AI outputs
   - Bias identification
   - Independent judgment
   - Ethical considerations
